Automatic Password locker and generator

This is an app that is capable of mastering, generating and storing user credentials ensuring good security, memory and acts as the accounts custodian

Description This is a terminal run python application that enables a user to auto master , lock and generate passwords from the credential Built By Vincent-Juma User freedom This will enable users to: * Copy credentials to the clipboard * Save login credentials * To create an account with the credentials * Generate new credentials for a new user
